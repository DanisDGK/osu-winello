# osu-winello
osu! installer for Linux with pre-packaged Wineprefix and patched wine-osu.

##Installation

#Prerequisites* 


Instructions:
```
git clone https://github.com/NelloKudo/osu-winello.git
cd osu-winello
chmod +x ./osu-winello.sh
sudo ./osu-winello.sh
```

You can now launch osu! with:
```osu-wine```
